# MIPS8ASM
Assembler for MIPS8 ISA.  This program takes as input a MIPS8 assembler file and generate a Verilog file, which represents a ROM memory whose contents is the program in Machine Language, ready to be executed.

#Example usage

The following assembly file:

```Assembly
Main :
    li r0, 0045
    li r1, 0xa
    li r2, 0xfb
    li r3, 0b11001011
Loop:
    add r0, r1
    addi r3, 1
    cmp r0, r3
    jz Main
    jmp Loop
```

Will be converted to the following Verilog file:

```Verilog
module InstructionMemory(
    input [7:0] address,
    output reg [15:0] data
);

    always @ (address)
    begin
        case (address)
            8'h00: data = 16'h485A;
            8'h01: data = 16'h4A14;
            8'h02: data = 16'h4DF6;
            8'h03: data = 16'h4F96;
            8'h04: data = 16'h0880;
            8'h05: data = 16'h5602;
            8'h06: data = 16'h6180;
            8'h07: data = 16'h6800;
            8'h08: data = 16'h8820;
            default: data = 16'h0000;
        endcase
    end
endmodule
```
